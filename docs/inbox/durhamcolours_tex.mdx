\subsection{Color Theory for Scientific Visualization}

\begin{axiom}[Visual Distinction]
Effective data visualization requires colors that are:
\begin{enumerate}
\item Distinguishable from each other
\item Accessible to colorblind viewers
\item Consistent across different media (print, screen)
\end{enumerate}
\end{axiom}

\begin{definition}[RGB Color Model]
Colors are specified as triples $(R, G, B)$ where each component is in $[0, 1]$:
\begin{equation}
\text{Color} = R \cdot \hat{r} + G \cdot \hat{g} + B \cdot \hat{b}
\end{equation}
Conversion from 8-bit values: $c_{norm} = c_{8bit} / 255$.
\end{definition}

\begin{theorem}[PANTONE to RGB Mapping]
Corporate color standards (PANTONE) are converted to RGB for digital display:
\begin{equation}
(R, G, B)_{RGB} = f_{display}(\text{PANTONE code})
\end{equation}
This conversion is display-dependent but standardized for consistency.
\end{theorem}

\subsection{Design Principles}

\begin{definition}[Color Harmony]
The Durham palette includes:
\begin{itemize}
\item Primary colors: Purple, Blue
\item Secondary colors: Red, Pink, Olive
\item Neutral colors: Grey, Black, Light Grey
\item Accent colors: Yellow, Light Blue, Light Purple
\end{itemize}
\end{definition}

\section{Color Accessibility}
%==============================================================================

\subsection{Colorblind Considerations}

The Durham palette includes colors that may be difficult to distinguish for colorblind viewers. Recommendations:
\begin{itemize}
\item Use \texttt{d\_blue} vs \texttt{d\_yellow} for maximum contrast
\item Combine color with line style (solid, dashed, dotted)
\item Add markers to distinguish data series
\end{itemize}

\subsection{Print Considerations}

Colors are designed to work in both color and grayscale print. The palette includes sufficient luminance variation for grayscale reproduction.

\section{Summary}
%==============================================================================

The \texttt{durhamcolours.py} module provides:

\begin{enumerate}
\item 12 named color variables for direct use in plotting
\item \texttt{cols} list for programmatic iteration
\item \texttt{colname} list for labels and legends
\item \texttt{help\_dcols()} function for visual reference
\end{enumerate}

Import methods:
\begin{lstlisting}
# Method 1: Import all colors
from durhamcolours import *

# Method 2: Import specific colors
from durhamcolours import d_purple, d_blue, d_red

# Method 3: Import module
import durhamcolours as dc
plt.plot(x, y, color=dc.d_purple)
\end{lstlisting}

The palette ensures consistent, professional styling across all ElecSus visualizations and publications.
