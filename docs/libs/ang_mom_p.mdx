## Description

Return J_+ matrix for angular momentum j. Calculates the + ladder operator (raising operator) for a given angular momentum.

Called by `ang_mom.py`.

## Dependencies

```python
from numpy import zeros, sqrt, arange
```

## Functions

### `jp(jj)`

Calculate the raising operator J_+ for angular momentum j.

**Arguments:**
- `jj` - Angular momentum quantum number (half-integer or integer)

**Returns:**
- J_+ matrix of dimension (2j+1) × (2j+1)

**Matrix elements:**
```
⟨j,m+1|J_+|j,m⟩ = √[j(j+1) - m(m+1)]
```

All other elements are zero.

## Source Code

```python showLineNumbers
from numpy import zeros, sqrt, arange

def jp(jj):
    b = 0
    dim = int(2*jj + 1)
    jp = zeros((dim, dim))
    z = arange(dim)
    m = jj - z
    while b < dim - 1:
        mm = m[b + 1]
        jp[b, b + 1] = sqrt(jj*(jj + 1) - mm*(mm + 1))
        b = b + 1
    return jp
```

## Usage Example

```python
import numpy as np
from ang_mom_p import jp

# Spin-1/2 raising operator
Sp = jp(0.5)
print("S_+ for j=1/2:")
print(Sp)
# [[0. 1.]
#  [0. 0.]]

# Angular momentum j=1 raising operator
J1p = jp(1)
print("\nJ_+ for j=1:")
print(J1p)
# [[0.         1.41421356 0.        ]
#  [0.         0.         1.41421356]
#  [0.         0.         0.        ]]

# The lowering operator is the transpose
Jm = Sp.T
print("\nS_- = (S_+)^T:")
print(Jm)
```

## Ladder Operator Properties

The raising operator J_+ acts on angular momentum states:
```
J_+ |j, m⟩ = √[j(j+1) - m(m+1)] |j, m+1⟩
```

The lowering operator J_- (obtained via transpose) acts as:
```
J_- |j, m⟩ = √[j(j+1) - m(m-1)] |j, m-1⟩
```

### Matrix Structure

J_+ has non-zero elements only on the upper diagonal:
- Element (m, m+1) = √[j(j+1) - m(m+1)]

Example for j = 1 (m = +1, 0, -1):
```
      |+1⟩  |0⟩   |-1⟩
⟨+1|  [ 0    √2    0  ]
⟨0|   [ 0    0     √2 ]
⟨-1|  [ 0    0     0  ]
```
