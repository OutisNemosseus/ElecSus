## Description

Variables used by the GUI. Contains formatting parameters, default values, parameter lists, and tooltip strings.

## GUI Layout Variables

```python
BtnSize = 30  # Button vertical size in pixels
```


## Usage

This module is primarily used internally by the ElecSus GUI. For programmatic access:

```python
from preamble import (
    element_list,
    D_line_list,
    defaultvals_main_parameterlist,
    OutputTypes
)

# Get available elements
print(f"Available elements: {element_list}")

# Get default temperature
temp_idx = main_parameterlist.index('Temperature')
print(f"Default temperature: {defaultvals_main_parameterlist[temp_idx]} Â°C")
```
