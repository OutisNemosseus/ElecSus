## Usage

```python
import matplotlib.pyplot as plt
import numpy as np
from durhamcolours import *

x = np.linspace(0, 10, 100)

plt.figure(figsize=(10, 6))
plt.plot(x, np.sin(x), color=d_purple, lw=2, label='sin(x)')
plt.plot(x, np.cos(x), color=d_blue, lw=2, label='cos(x)')
plt.plot(x, np.sin(x)**2, color=d_red, lw=2, label='sinÂ²(x)')
plt.fill_between(x, 0, np.sin(x), color=d_lightpurple, alpha=0.3)
plt.legend()
plt.show()
```

## Functions

### `help_dcols()`

Display a plot showing all available colours with their names.

```python
from durhamcolours import help_dcols
help_dcols()
```

Saves `durhamcolors.png` and `durhamcolors.pdf` to the current directory.
