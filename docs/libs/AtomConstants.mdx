## Description

Atom/transition related constants. Hyperfine constants are in units of MHz. Nuclear spin g-factors are those corresponding to the Bohr magneton.

Called by `EigenSystem.py`

## Dependencies

```python
from numpy import pi, sqrt
from FundamentalConstants import *
```

## Atom Classes

### `IdealAtom`
Constants for an ideal atom with no hyperfine structure, and only electron spin.

| Property | Value | Description |
|----------|-------|-------------|
| `I` | 0 | Nuclear spin |
| `As` | 0 | Ground state hyperfine constant (MHz) |
| `gI` | 0 | Nuclear spin g-factor |
| `mass` | 85*amu | Atomic mass |

### `Rb85`
Constants relating to the rubidium-85 atom.

| Property | Value | Description |
|----------|-------|-------------|
| `I` | 2.5 | Nuclear spin |
| `As` | 1011.910813 | Ground state hyperfine constant (MHz) |
| `gI` | -0.00029364 | Nuclear spin g-factor |
| `mass` | 84.911789732*amu | Atomic mass |
| `FS` | 7.123e6 | Fine-structure splitting (MHz) |

### `Rb87`
Constants relating to the rubidium-87 atom.

| Property | Value | Description |
|----------|-------|-------------|
| `I` | 1.5 | Nuclear spin |
| `As` | 3417.341305452145 | Ground state hyperfine constant (MHz) |
| `gI` | -0.0009951414 | Nuclear spin g-factor |
| `mass` | 86.909180520*amu | Atomic mass |
| `FS` | 7.123e6 | Fine-structure splitting (MHz) |

### `Cs`
Constants relating to the caesium-133 atom.

| Property | Value | Description |
|----------|-------|-------------|
| `I` | 3.5 | Nuclear spin |
| `As` | 2298.1579425 | Ground state hyperfine constant (MHz) |
| `gI` | -0.00039885395 | Nuclear spin g-factor |
| `mass` | 132.905451931*amu | Atomic mass |

### `K39`, `K40`, `K41`
Constants for potassium isotopes.

### `Na`
Constants for sodium-23.

Constants relating to the rubidium D2 transition.

| Property | Value | Description |
|----------|-------|-------------|

### `CsD1Transition`, `CsD2Transition`

### `KD1Transition`, `KD2Transition`

### `NaD1Transition`, `NaD2Transition`
Constants for sodium D-lines.

## Isotope-Transition Classes

These classes provide hyperfine coupling constants for specific isotope-transition combinations.

### `Rb85_D1`, `Rb85_D2`
| Property | D1 Value | D2 Value | Description |
|----------|----------|----------|-------------|
| `Ap` | 120.640 | 25.038 | Excited state A coefficient (MHz) |
| `Bp` | 0.0 | 26.011 | Excited state B coefficient (MHz) |
| `IsotopeShift` | 21.624 | 21.734 | Isotope shift (MHz) |

### `Rb87_D1`, `Rb87_D2`
| Property | D1 Value | D2 Value | Description |
|----------|----------|----------|-------------|
| `Ap` | 406.147 | 84.7185 | Excited state A coefficient (MHz) |
| `Bp` | 0.0 | 12.4965 | Excited state B coefficient (MHz) |
| `IsotopeShift` | -56.077 | -56.361 | Isotope shift (MHz) |

Similar classes exist for `Cs_D1`, `Cs_D2`, `K39_D1`, `K39_D2`, `K40_D1`, `K40_D2`, `K41_D1`, `K41_D2`, `Na_D1`, `Na_D2`.

## Usage Example

```python
from AtomConstants import Rb87, RbD2Transition, Rb87_D2

# Get Rb-87 properties
print(f"Rb-87 nuclear spin: {Rb87.I}")
print(f"Rb-87 mass: {Rb87.mass} kg")

# Get D2 transition properties
print(f"D2 wavelength: {RbD2Transition.wavelength * 1e9:.3f} nm")
print(f"D2 linewidth: {RbD2Transition.NatGamma} MHz")

# Get isotope-specific hyperfine constants
print(f"Rb-87 D2 Ap: {Rb87_D2.Ap} MHz")
```
